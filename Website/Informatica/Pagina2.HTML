De Geschiedenis van JavaScript

De geschiedenis van JavaScript is best wel verwoven met de andere programeertaal 'Java'. JavaScript lijkt namelijk niet alleen in naam op Java, maar ook qua syntaxis (taalregels). En daar is een hele goede reden voor als je kijkt naar de geschiedenis. JavaScript is namelijk ontstaan toen Java populair werd. Da taal Java is ontwikkeld door het elektronisch bedrijf Sun Microsystems. In 1995 werd Java heel populair, maar omdat de taal zo complex was, waren er maar weinig mensen die er goed mee om konden gaan. Het bedrijf Netscape (webbrowser) zag dat er behoefte was aan een taal met veel van de mogelijkheden van Java, maar die wel eenvoudig te leren was en die verder direct door de browser uitgevoerd kon worden. In de tweede helft van 1995 begonnen Netscape en Sun een samenwerking die de creatie van JavaScript tot gevolg had. JavaScript heette eerst "Mocha" en later werd het "Livescript" genoemd. Toen de samenwerking tussen Netscape en Sun eenmaal een feit was, ontstond de uiteindelijke naam: JavaScript.

Hieronder zie je een tabel over de een aantal belangrijke punten in de geschiedenis van Javascript.
