Wat is JavaScript

JavaScript, afgekort als JS, is een programeertaal. Het is zelfs de populairste programeertaal ter wereld. JavaScript is een script-taal om een webpagina interactiever te maken. Een scripttaal is een programma dat geschikter is voor het schrijven van kleine programmaatjes om veel voorkomende of kleine taken te verrichten.
